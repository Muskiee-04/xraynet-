# XrayNet+

Offline-First, Explainable AI Framework for Thoracic Disease Screening in Low-Resource Clinical Settings

##  Overview

XrayNet+ is an innovative AI-powered diagnostic assistant designed specifically for low-resource healthcare settings. It provides accurate, explainable thoracic disease screening from chest X-rays while operating entirely offline.

###Key Features

- ** Offline-First Operation** - Functions without internet connectivity
- ** Explainable AI** - Visual heatmaps show the AI's reasoning using Grad-CAM++
- ** Data Privacy** - All data processed locally, no cloud dependencies
- ** Optimized Performance** - Pruned and quantized models for low-end hardware
- ** Clinical Workflow Integration** - Streamlit-based interface with automated reporting
- ** Containerized Deployment** - Docker support for easy installation
